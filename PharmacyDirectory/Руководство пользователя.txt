Руководство пользователя программы управления каталогом лекарств
1. Описание программы:
Программа позволяет пользователю вести каталог лекарств, включая добавление, редактирование, удаление, сортировку, поиск, а также сохранение данных в файл и загрузку из него.

2. Основные функции:
Импорт данных из файла

Выберите пункт "1" в меню, введите название файла (с расширением .txt), из которого хотите загрузить данные. Программа загрузит данные о лекарствах в каталог.
Ввод информации о новом лекарстве

Выберите пункт "2", чтобы добавить новое лекарство в каталог. Программа запросит следующие данные:
Название лекарства
Название фирмы-производителя
Страна-производитель
Назначение лекарства
Требуется ли рецепт
Форма выпуска
Введенные данные будут проверены на корректность.
Редактирование лекарства

Выберите пункт "3" для редактирования лекарства. Введите название лекарства, которое хотите изменить, после чего можно изменить каждое из полей.
Удаление лекарства

Выберите пункт "4", чтобы удалить лекарство из каталога. Введите название лекарства, и оно будет удалено.
Просмотр всех лекарств

Выберите пункт "5" для отображения всех лекарств в каталоге в виде таблицы.
Поиск лекарства по названию

Выберите пункт "6", чтобы найти лекарство по его названию.
Список лекарств одной фирмы

Выберите пункт "7", чтобы получить список всех лекарств, произведенных одной фирмой. Введите название фирмы для отображения результатов.
Поиск болеутоляющих средств без рецепта

Выберите пункт "8", чтобы найти болеутоляющие средства, которые не требуют рецепта.
Сортировка лекарств

Выберите пункт "9", чтобы отсортировать каталог лекарств. Программа предложит ввести поле для сортировки:
name (по названию)
company (по фирме)
country (по стране)
purpose (по назначению)
form (по форме выпуска)
Сохранение данных в файл

Выберите пункт "10", чтобы сохранить текущие данные в новый файл. Программа предложит ввести имя файла с расширением .txt. Если файл уже существует, будет предложено его перезаписать.
О программе

Выберите пункт "11", чтобы получить информацию о программе.
Руководство пользователя

Выберите пункт "12", чтобы загрузить и просмотреть руководство пользователя.
Выход

Выберите пункт "13", чтобы выйти из программы.
3. Рекомендации по работе с программой:
Корректность ввода: программа проверяет вводимые данные (например, названия должны начинаться с заглавной буквы и не содержать специальных символов).
Формат файлов: все данные сохраняются и загружаются в файлы формата .txt, разделенные символом ;.
